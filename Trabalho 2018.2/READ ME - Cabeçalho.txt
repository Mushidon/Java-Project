Trabalho de simulação de uma empresa de lógistica com interface gráfica para mostrar o menor preço de um transporte realizado por um Veículo autonomo.
A persistência foi realizada usando o HSQLDB do JDBC e a tabela deve ser criada da seguinte forma: 

SQL DAS TABELAS
				  	
					create table AUTONOMOS (	
					
							matricula VARCHAR(50),		
							capacidade double,
							latitude int,
							longitude int,
							id int
						
					);

Ao rodar o projeto, apareceram 3 telas, a primeira corresponde ao usuário colocar as coordenadas de coleta e de entrega do produto juntamente com o seu peso. 
As coordenadas tanto de coleta quanto de destino serão capturas e jogadas no método que irá pegar o veiculo com o menor custo dentre aqueles que estão cadastrados no banco de dados. 
Ao confirmar será exibido na segunda tela com a matricula do veiculo que possui o menor preço dentre os cadastrados no banco juntamente com o custo do seu transporte. 
Após confirmar o transporte, as coordenadas do veiculo serão atualizadas para as coordenadas que o usuário colocou como destino dentro do banco de dados, caso o usuário não confirme o update não é feito.
