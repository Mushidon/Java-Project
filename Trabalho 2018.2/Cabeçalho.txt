Trabalho de simulação de uma empresa de lógistica com interface para o menor preço de um transporte por um veiculoautonomo.
A persistência foi realizada usando o HSQLD do JDBC e a tabela deve ser criada da seguinte forma: 

SQL DAS TABELAS
				  	
					create table AUTONOMOS (	
					
							matricula VARCHAR(50),		
							capacidade double,
							latitude int,
							longitude int,
							ID int
						
					);

Ao rodar o projeto, apareceram 3 telas, a primeira corresponde ao usuário colocar as coordenadas de coleta e de entrega do produto juntamente com o seu peso. 
Ao confirmar será exibido na segunda tela com a matricula do veiculo que possui o menor preço dentre os cadastrados no banco juntamente com o custo do seu transporte. 
Após confirmar o transporte, as coordenadas do produto serão atualizadas para as coordenadas que o usuário colocou como destino.
